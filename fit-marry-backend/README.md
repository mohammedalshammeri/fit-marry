# Fit-Marry Backend

Backend API for the Fit-Marry platform (NestJS + Prisma + PostgreSQL).

## Requirements
- Node.js 18+
- PostgreSQL (Neon or local)

## Setup
1) Install deps
2) Configure environment (.env)
3) Generate Prisma client and run migrations

## Scripts
- `start:dev`
- `build`
- `test`

## Notes
- Admin APIs require admin JWT.
- Swagger docs at `/docs`.
